## 🚀 Quick Start (Get Running in 10 Minutes)

### Step 1: Extract & Install
```bash
# Extract the ZIP file
cd medconnect-ai

# Install dependencies
npm install
```

### Step 2: Get FREE Gemini API Key
1. Visit: https://makersuite.google.com/app/apikey
2. Sign in with Google
3. Create API key
4. Copy the key

### Step 3: Setup Environment
```bash
# Copy example file
cp .env.example .env

# Edit .env and add your key:
VITE_GEMINI_API_KEY=AIzaSyC_your_actual_key_here
```

### Step 4: Start App
```bash
npm run dev
```

### Step 5: Test Features
1. Go to http://localhost:3000
2. Click "Prescriptions"
3. Upload a prescription image
4. Watch the magic! ✨

## 💡 Pro Tips

### 1. Test Before Demo
- Upload 5-10 prescriptions beforehand
- Check if OCR works well
- Verify AI extraction accuracy

### 2. Prepare Good Images
- Clear, well-lit photos
- Straight angle (not tilted)
- Readable text
- Standard prescription format

### 3. Handle Errors Gracefully
- App shows clear error messages
- Retry mechanism built-in
- Fallback to manual entry

### 4. Know Your Limits
- 15 API calls per minute (plenty!)
- Don't spam uploads during demo
- One prescription = 1 API call
